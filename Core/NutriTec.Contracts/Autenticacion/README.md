# Autenticacion

Esta carpeta contiene DTOs públicos para operaciones iniciales de autenticación.

Responsabilidades principales:

- Definir solicitudes de login y registro.
- Definir respuestas seguras para usuarios autenticados o registrados.
- Evitar exponer contraseñas, hashes o detalles internos de persistencia.

Restricciones:

- No implementa JWT.
- No implementa hashing.
- No define tablas ni repositorios concretos.
