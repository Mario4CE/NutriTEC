# Common

Esta carpeta contiene elementos transversales compartidos por los casos de uso.

Responsabilidades principales:

- Definir excepciones de aplicación.
- Centralizar tipos reutilizables de la capa Application.
- Facilitar manejo consistente de errores esperados.

Restricciones:

- No debe contener lógica específica de un controller.
- No debe depender de infraestructura.
- No debe incluir utilidades sin uso claro en la aplicación.
