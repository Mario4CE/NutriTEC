# Productos

Esta carpeta contiene entidades de dominio relacionadas con productos alimenticios.

Responsabilidades principales:

- Representar un producto con información nutricional básica.
- Mantener estado de aprobación y fecha de creación.
- Servir como modelo de negocio para los casos de uso de productos.

Restricciones:

- No debe actuar como DTO público.
- No debe contener rutas HTTP.
- No debe incluir lógica de persistencia concreta.
