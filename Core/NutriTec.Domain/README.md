# NutriTec.Domain

Proyecto que contiene entidades del dominio de NutriTEC.

Responsabilidades principales:

- Representar conceptos centrales del negocio.
- Mantener modelos independientes de controllers, DTOs e infraestructura.
- Servir como núcleo reutilizable por Application.

Restricciones:

- No debe depender de ASP.NET Core.
- No debe depender de Entity Framework Core o MongoDB como detalle obligatorio.
- No debe contener contratos públicos para el frontend.
